## 4. Что нужно от заказчика (вводные)

**Блокирует выход (нужно сейчас):**
1. **WhatsApp менеджера** — номер в формате `77XXXXXXXXX`
2. **Телефон** для звонка (отображение + для `tel:`)
3. **Email**
4. **Город**
5. **Instagram** (ссылка + @-имя)
6. **Ссылка на Telegram-бота** (если бот ещё не готов — можно временно вести на менеджера/канал)
7. **Домен** и доступ к хостингу

**Для Фазы 2:**
8. **Фото товаров** (постельное, полотенца, халаты, тапочки, одноразовые наборы; желательно в гостиничной подаче)

**Для Фазы «П» (бот):**
9. **Цены по стандартным размерам** постельного (1,5 / 2-сп / Евро / Семейный) — §8.5
10. **Прайс одноразовых** (есть) + **правила расчёта** (объём, скидки, минимальный заказ)

**Юридическое (желательно):**
11. Реквизиты/название юрлица для футера; нужна ли «Политика конфиденциальности».

## 6. План деплоя

1. Заполнить `CONFIG` реальными контактами.
2. Финальная проверка (desktop + mobile + reduced-motion).
3. Залить `index.html` + `assets/` на статический хостинг.
   - Варианты: GitHub Pages · Netlify · Vercel · Cloudflare Pages · обычный shared-хостинг.
4. Подключить домен, проверить HTTPS.
5. Прогнать ссылки (WhatsApp, tel, email, Telegram), отправить тестовую заявку.

## 7. Риски и открытые вопросы

| # | Вопрос | Решение / статус |
|---|---|---|
| 1 | Уведомление в Telegram-бот из статики | Прямой Bot API в браузере небезопасен (токен утечёт). Нужен промежуточный endpoint → `CONFIG.leadEndpoint`. Пока — канал WhatsApp. |
| 2 | Нет фото | v1 — премиум-плейсхолдеры; слоты готовы под замену без переверстки. |
| 3 | Сбор ПДн формой | Сайт не хранит, но передаёт телефон в WhatsApp. Для РК желательна короткая политика/согласие у кнопки. Уточнить. |
| 4 | Бренд: «Comfy Eden» vs «EDEN HoReCa» | Решено: вордмарк Comfy Eden. Если поменяется — правка в 1 строке HTML + favicon. |
| 5 | Готовность бота | Лендинг не блокируется ботом; ссылку на бота можно подставить позже. |
