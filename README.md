# Comfy Eden — лендинг

Premium B2B-лендинг гостиничного текстиля и одноразовых принадлежностей для HoReCa (Казахстан).
Статический сайт: чистый HTML + CSS + JS, без сборки и зависимостей. Хостится где угодно.

Реализовано по [SPEC.md](SPEC.md), v1.5.

## Структура

```
index.html              — все секции (hero, продукция, для кого, почему мы, как работаем, форма, контакты)
assets/css/styles.css   — стили, палитра «песок + изумруд», motion
assets/js/main.js       — конфиг контактов, motion, sticky-storytelling, форма → WhatsApp
README.md               — этот файл
SPEC.md                 — техзадание
```

## Как открыть

Просто откройте `index.html` в браузере — двойной клик.

Либо локальный сервер (чтобы корректно работали все пути):

```powershell
# из папки проекта
python -m http.server 5500
# затем откройте http://localhost:5500
```

## ⚙️ Что заполнить перед публикацией

Все контакты — **в одном месте**: объект `CONFIG` в начале [assets/js/main.js](assets/js/main.js).
Найдите пометки `// TODO` и подставьте реальные данные:

| Поле | Что это |
|------|---------|
| `whatsappNumber` | Номер WhatsApp менеджера. **Без `+`**, только цифры: `77XXXXXXXXX` |
| `phoneDisplay` / `phoneHref` | Телефон: как показывать / для ссылки `tel:` |
| `email` | Почта |
| `city` | Город |
| `instagramUrl` / `instagramHandle` | Ссылка и @-имя Instagram |
| `telegramBotUrl` | Ссылка на Telegram-бота (каталог, прайс, КП) |
| `leadEndpoint` | *(необязательно)* URL, который перешлёт заявку в бот/CRM |

> Контакты в HTML — это запасные плейсхолдеры на случай отключённого JS. JS подставляет данные из `CONFIG` поверх них. Менять достаточно `CONFIG`.

## Как работает форма заявки

1. Проверяет обязательные поля (имя, город, телефон).
2. Собирает готовое сообщение и **открывает WhatsApp** менеджеру (`whatsappNumber`) — основной канал (§8.2 SPEC).
3. Если задан `leadEndpoint` — дополнительно шлёт заявку туда JSON-ом (для уведомления в Telegram-бот). Это best-effort и не блокирует WhatsApp.
4. Сайт **ничего не хранит** — данные только передаются (§8.1).

### Уведомление в Telegram-бот

Прямой вызов Telegram Bot API из браузера небезопасен (токен утечёт). Когда бот будет готов, заведите простой промежуточный endpoint (serverless-функция / webhook), который примет JSON заявки и отправит его боту, и впишите его URL в `CONFIG.leadEndpoint`. До этого заявки идут через WhatsApp.

## 📷 Слоты под фото (фаза с фото)

Сейчас фото нет — используются премиум CSS-плейсхолдеры (§5.3 SPEC). Места под реальные фото:

- **Hero** — карточки `.hero__card` (`index.html`, блок `.hero__art`). Слот фронт-карточки 4:5.
- **Продукция (storytelling)** — `.story__media` в sticky-панели. Можно заменить иконку категории на фото.

Чтобы подключить фото: положите файлы в `assets/img/`, замените текстурный `<div class="ph …">` на `<img>` с тем же размером слота. Вёрстку менять не нужно.

## Дизайн

- **Палитра:** песок/айвори + изумруд; брасс только как акцент уровня «Премиум».
- **Шрифты:** Playfair Display (заголовки) + Manrope (текст), Google Fonts, с поддержкой кириллицы.
- **Motion (§6.2):** reveal при скролле · sticky-сторителлинг категорий · hover-микроанимации. Уважается `prefers-reduced-motion`.

## Деплой
