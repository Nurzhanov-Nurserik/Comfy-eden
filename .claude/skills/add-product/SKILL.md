# Добавление товара — Comfy Eden

Данные товаров = контент в коде (§8.3), не БД. Цены сюда НЕ входят (§8.5).

## Поля карточки (§8.3)
- Название
- Категория (постельное / подушки и одеяла / наматрасники-чехлы / полотенца / халаты / тапочки / одноразовое)
- Размер (для постельного — 1,5-спальный / 2-спальный / Евро / Семейный или нестандарт)
- Материал, плотность
- Цвет
- Уровень качества: Эконом / Стандарт / Премиум (§5.4)
- Краткое описание — ценность, а не только название (§4.3)
- Слот под фото (зарезервировать; фото может появиться позже)

## Шаги
1. Определи секцию: сейчас ассортимент раскрыт в sticky-сторителлинге `#products` в `index.html`.
2. Добавь разметку в стиле существующих карточек/шагов: те же классы; чипы уровней — `<span class="chip" data-level="...">`.
3. Фото: используй плейсхолдер `.ph` или `<img>` из `assets/img/`, не ломая вёрстку слота.
4. Описание — короткое, продающее ценность; без цен.
5. Передай результат на `design-qa`.

## Правила
- Не публикуй цены.
- Уровень качества — обязателен.
- Тексты — в тоне проекта (см. субагент `ru-copywriter`).

## Пример
Добавили в `#products` позицию «Наматрасник на резинке»: категория наматрасники-чехлы, стандартные размеры, уровни Стандарт/Премиум через `<span class="chip" data-level>`, слот под фото `.ph`, описание без цены. Передали на `qa-review`.

Связано: субагенты `frontend-dev`, `ru-copywriter`; документы `SPEC.md` §5.4, §8.3.
