# Новая секция (скилл агента `frontend-dev`)

Верстаем новый блок так, будто он был тут с самого начала: те же классы, палитра песок+изумруд, сдержанный motion, адаптив.

## Шаги
1. Выбери место в `index.html` и возьми похожую существующую секцию как образец.
2. Свёрстай в стиле проекта: `.container` → `.section`, `.eyebrow` + `.section__title`, карточки `.card` / сетка `.grid--3`, BEM-классы.
3. Добавь появление `data-reveal` (+ `data-reveal-delay` для стаггера) и hover; ничего нового в JS без нужды.
4. Проверь брейкпоинты 900/600 и `prefers-reduced-motion`, затем передай на `qa-review`.

## Проверка
- [ ] Классы и отступы консистентны с проектом.
- [ ] Адаптив на 900/600 не ломается; reduced-motion ок.
- [ ] Нет ошибок в консоли; цены не добавлены (§8.5).

## Пример
Добавили секцию «Отзывы» после «Как работаем»: сетка `.grid--3`, карточки в стиле `.card`, заголовок через `.eyebrow` + `.section__title`, reveal со стаггером `data-reveal-delay`. Прогнали `qa-review` — чисто.

Связано: агент `frontend-dev`; скилл `qa-review`; файлы `index.html`, `assets/css/styles.css`, `SPEC.md` §7; `TASK.md` эпики A, F.
