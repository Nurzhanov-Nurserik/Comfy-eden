# Генерация КП (PDF) — Comfy Eden

КП формирует бот или менеджер, НЕ сайт (§8.4). Этот скилл собирает PDF-документ.

## Вход (данные КП — §8.4)
- **Клиент:** компания, контактное лицо, город.
- **Позиции:** наименование, категория, размер, материал/плотность, уровень качества (Эконом/Стандарт/Премиум), кол-во, цена за единицу, сумма.
- **Итог:** сумма, скидка (если есть), с НДС / без НДС.
- **Условия:** оплата, поставка, сроки, **срок действия КП**.
- **Реквизиты** Comfy Eden.

## Шаги
1. Проверь, что цены заданы (стандартные размеры — из прайса бота; нестандарт — расчёт менеджера). Без цен КП не собирай.
2. Составь таблицу позиций, посчитай суммы и итог.
3. Сгенерируй PDF (используй скилл `pdf`): шапка с брендом Comfy Eden (палитра песок+изумруд), таблица позиций, итог, условия, срок действия, контакты.
4. Имя файла: `KP_Comfy-Eden_<клиент>_<дата>.pdf`.

## Правила
- Срок действия КП — обязателен.
- Уровень качества указывай по каждой позиции.
- Только подтверждённые цены и условия — ничего не выдумывай.

## Пример
Клиент запросил постельное «Стандарт» на 40 номеров (2-спальный) + одноразовые наборы. Цены взяты из прайса бота, посчитан итог, собран `KP_Comfy-Eden_Hotel-Astana_2026-06-12.pdf` со сроком действия 14 дней.
