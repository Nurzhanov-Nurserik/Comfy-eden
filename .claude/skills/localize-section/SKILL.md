# Локализация на казахский (скилл агента `kz-localizer`)

Переводим контент на казахский так, чтобы тон остался премиальным и деловым, а вёрстка не поехала.

## Шаги
1. Возьми RU-текст нужной секции из `index.html`.
2. Переведи на KZ в деловом регистре естественно (не дословно); бренд `Comfy Eden`, имена и контакты не трогай.
3. Сверь термины с глоссарием (бельё, махра, плотность, уровни Эконом/Стандарт/Премиум); пополни глоссарий при необходимости.
4. Внеси в KZ-версию, сохрани классы; проверь длину строк → передай на `qa-review`.

## Проверка
- [ ] Тон и смысл сохранены; терминология единая.
- [ ] Бренд/контакты/числа корректны; цен не добавлено.
- [ ] Длина текста не ломает вёрстку.

## Пример
Перевели hero на казахский, обновили глоссарий (наматрасник, плотность), заголовок влез в `.hero__title` без переполнения. Прогнали `qa-review` на KZ-версии — чисто.

Связано: агент `kz-localizer`; скилл `qa-review`; файлы `index.html`, `SPEC.md` §13, `TASK.md` G5.
